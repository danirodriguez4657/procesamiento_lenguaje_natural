# Procesamiento de Lenguaje Natural — Embeddings en Español

Laboratorio de NLP enfocado en representación semántica de texto en español usando técnicas de embeddings.

## Descripción

Análisis de frases en español con contenido mixto (texto + emojis) para explorar cómo los modelos de embeddings capturan el significado semántico en lenguaje informal y coloquial.

## Contenido

- `dataset_lab3.csv` — Dataset de frases en español con emojis
- `Laboratorio_III_Embeddings.ipynb` — Notebook con implementación de embeddings

## Técnicas utilizadas

- Word embeddings (Word2Vec / FastText / GloVe)
- Análisis de similitud semántica
- Visualización de espacios vectoriales

## Tecnologías

Python · scikit-learn · NLTK · Gensim · Matplotlib
